## Why elm-pages?

elm-pages gives you type-safe static site generation with the full power of Elm. Combine it with Tailwind CSS and a component library to build fast, beautiful sites.

<feature-grid columns="3">

<feature title="Type-safe" icon="✓">

Every route, page, and component is checked at compile time. No runtime surprises.

</feature>

<feature title="Markdown + Components" icon="◆">

Write in Markdown and drop in rich UI components using simple HTML tags.

</feature>

<feature title="Tailwind CSS" icon="✦">

Utility-first styling with full Tailwind v4 support baked in via the Vite plugin.

</feature>

</feature-grid>
